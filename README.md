# Sparta2DTopDown
유니티 강의 기반으로 캐릭터 만들기, 캐릭터 이동, 방 만들기 구현<br>
카메라 따라가기는 구글 서치해 나온 코드 이용

- **필수 요구 사항**
    - [x]  캐릭터 만들기
    - [x]  캐릭터 이동
    - [x]  방 만들기
    - [x]  카메라 따라가기
    - [ ]  캐릭터 애니메이션 추가
    - [ ]  이름 입력 시스템
    - [ ]  캐릭터 선택 시스템
- **선택 요구 사항**
    - [ ]  시간 표시
    - [ ]  인게임 이름 바꾸기
    - [ ]  참석 인원 UI
    - [ ]  인게임 캐릭터 선택
    - [ ]  NPC 대화
